 
# AI-Driven Strategic HR Platform

**An Integrated AI Solution for Recruitment, Assessment, and Document Analysis**

---

## Project Description

The **AI-Driven Strategic HR Platform** is an AI-powered system designed to optimize Human Resources (HR) processes. It addresses common HR challenges such as manual candidate screening, subjective evaluations, and unstructured data management by leveraging **Artificial Intelligence** technologies.

### Key Modules

1. **AI HR Agent**  
   - An intelligent assistant that analyzes resumes and answers natural language queries.
   - Uses **SentenceTransformers**, **FAISS**, **LangChain**, and a **LoRA fine-tuned LLaMA 3 model** for context-aware candidate matching.

2. **Quiz Generator**  
   - Automatically generates quizzes and assessments based on job descriptions or candidate profiles.
   - Ensures fairness and standardization in candidate evaluation.

3. **OCR Document Analyzer**  
   - Extracts structured information from scanned resumes, certificates, and forms.
   - Uses **EasyOCR** and **OpenCV** to convert unstructured images into usable data.

### Objectives

- Automate HR tasks like CV parsing and candidate matching.
- Improve decision-making accuracy with context-aware AI.
- Ensure fairness and consistency in candidate evaluation.
- Digitize HR workflows for efficiency and transparency.

---

This platform demonstrates how AI can **empower recruiters**, streamline HR processes, and create **data-driven, fair, and efficient recruitment solutions**.
